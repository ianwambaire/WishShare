# Birthday Wishlist Application

This is an iOS mobile application built using Swift and Storyboard that allows users to create personal wishlists, view friends’ birthday wishlists, contribute towards gifts as a group, and receive reminders for upcoming birthdays.

The application focuses on simplifying birthday gift planning by enabling social collaboration, wishlist sharing, and pooled contributions toward gifts.

---

## Application Features

The application supports social interactions between users and provides tools for birthday planning and group gifting.

### User and Social Features
- Create and manage a personal user profile  
- Send friend requests to other users  
- Accept or reject incoming friend requests  
- View and manage a list of friends  

### Birthday Management
- View upcoming birthdays of friends  
- Receive reminders when a friend’s birthday is approaching  

### Wishlist Management
- Create a personal wishlist  
- Add items to a wishlist  
- View friends’ wishlists  

### Group Gifting
- Contribute money towards a gift as a group  
- Track individual contributions from each friend  
- View progress toward achieving the gift goal  

---

## Technology Stack

- Programming Language: Swift  
- User Interface: Storyboard  
- Platform: iOS  
- Architecture: MVC  
- Development Environment: Xcode

## Installation

### Clone the Repository

Use GitHub to clone the repository locally.

### Open in Xcode

1. Open the cloned project folder.
2. Double-click the `.xcodeproj` file to open it in Xcode.
3. Select an iOS simulator or a connected physical device.
4. Run the application using **Command + R**.
